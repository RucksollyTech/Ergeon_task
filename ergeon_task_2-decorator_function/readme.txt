Task 2
Write a decorator in python that will count how many times the decorated function was called. It should print
the number every time the decorated function is executed. Each function should be counted separately.

The function is created inside the folder "Utils -> function_call_count" 
and named "count_calls".

The folder "myFunction" contains example usage of this function "count_calls".

To run this file

cd path/to/ergeon_task_2
python -m myFunction.func_file
